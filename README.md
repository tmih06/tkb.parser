# tkb.parser

Forked from [dut.tkb.parser](https://github.com/michioxd/dut.tkb.parser)

## Development

```sh
# Clone the repository
git clone https://github.com/tmih06/tkb.parser.git

# Install dependencies
cd tkb.parser
pnpm i

# Run the development server
pnpm dev
```
